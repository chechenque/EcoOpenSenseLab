Especificaciones Técnicas

  1. Alimentación: Conecta el pin VCC del sensor al pin de 3.3V o 5V del microcontrolador.
  2. Tierra: Conecta el pin GND del sensor al pin de tierra GND del microcontrolador.
  3. Señal/Datos: Conecta el pin de Datos/SDA al pin digital recomendado (ver especificaciones técnicas del sensor).
  4. Resistencia (Opcional): Para sensores DHT22, se aconseja una resistencia pull-up de 10kΩ entre VCC y la línea de datos si el módulo no viene pre-ensamblado en placa.

* Nota: Estas conexiones forman parte de la etapa de Recolección y aseguran la correcta lectura física del ecosistema.

Copia este sketch en el Arduino IDE para programar tu microcontrolador y transmitir datos en formato serial legible para su posterior Organización.
